Cardano HTML DApp

Welcome to the Cardano HTML DApp example! This page showcases how to integrate `cardano-connect.js` into your HTML-based DApp using the cardano-connect-with-wallet-core library.

Getting Started

To get started, import the latest core `bundle`, `cardano-connect.js`, and generate a parent node to initalize the component:

<head> <script src="https://cardano-foundation.github.io/cardano-connect-with-wallet/bundle-latest/index.js"></script> <script src="https://cardano-foundation.github.io/cardano-connect-with-wallet/cardano-connect-latest/index.js"></script> <script> window.onload = () => { const container = document.getElementById('cardano-connect-container') cardanoConnectWallet = new CardanoConnectWallet(container) } </script> </head> <body> <div id="cardano-connect-container" /> </body>
